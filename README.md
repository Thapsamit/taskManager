# Heroku deployement
##  Step -1 Download heroku cli

## Step - 3 Create new App from heroku website

        -> heroku git:remote -a <nameOfApp>

        write in cmd
        -> heroku config
        -> heroku config:set API_KEY = "Your Api key"
